Online Verbal Counting Simulator

This simulator will allow to train oral counting skills for both children and adults in a simple and convenient form. In the simulator, you can select tasks for various arithmetic operations: multiplication, division, addition and subtraction. A combined mode is also available, which includes all of the above.

You can set the level of difficulty of the tasks: easy, intermediate, hard and very hard. The sections on multiplication and division also have table mode, which includes tasks based on multiplication and division tables.

Additional settings can be used to create optimal usability conditions:

  • you can set the number of tasks that will be proposed for solving
  • displaying the task counter will help to monitor the number of tasks remaining, as well as the number of correct and incorrect answers
  • the timer makes it possible to limit the amount of time for solving tasks
  • when choosing the option for showing hints, three possible solutions will be displayed, one of which is correct
  • finally, you can configure to show the correct answer if you make a mistake

To summarize, this simulator is an effective and convenient tool for training oral counting. It can be useful both as a game for children in the educational process, and as a puzzle for adults who want to improve their oral counting skills or spend free time for the benefit of the mind.
